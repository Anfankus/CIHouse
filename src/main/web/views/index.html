CIHouse新三板企业征信平台

数据全生命周期产品线

  • 数据获取
    数据获取

    使用Jsoup作为页面解析工具，从各个相关征信公示网站中获取到所需信息，并存储到数据库中，为大数据项目的后续工作奠定基础。

  • 数据处理
    数据处理

    使用Hadoop作为集群环境，从数据库中将原始数据整理归类，再将处理好的数据存入Hbase数据库中。

  • 数据存储
    数据存储

    使用Hadoop顶级项目Hbase作为数据存储，并使用公司代码作为行键，加快了搜索速度，丰富了数据存储的意义。

  • 数据展示
    数据展示

    使用html、css、JavaScript进行编写前端页面。数据展示部分详细美观的呈现了公司信息、公司对比、统计信息等功能，从而为用户呈现了数据结果。
